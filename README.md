gh-mv-issues
==========

`gh-mv-issues` moves the issues of a `github` repository to another repository.

## Installation

```sh
$> go get github.com/sbinet/gh-mv-issues
```

## Documentation

```sh
$> gh-mv-issues -help
```

## Example

```sh
$> gh-mv-issues github.com/user/my-repo github.com/our-organization/repo
```
